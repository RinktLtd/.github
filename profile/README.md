# Rinkt

**Developer-first automation — browser, desktop, Office & API workflows with durable execution.**

Rinkt is a full-stack automation platform that lets developers build and orchestrate workflows across browsers, desktop applications, Microsoft Office, and APIs — with built-in reliability, automatic retries, and state persistence.

## Repositories

| Repo | Description |
|---|---|
| [**runner**](https://github.com/RinktLtd/runner) | Cross-platform automation runner (Go / Wails / Playwright) |
| [**dispatcher**](https://github.com/RinktLtd/dispatcher) | Workflow task routing and scheduling |
| [**python-interpreter**](https://github.com/RinktLtd/python-interpreter) | Embedded Python interpreter for workflow scripting |

## Get started

Visit [rinkt.com](https://rinkt.com) to learn more, or [explore the Runner repo](https://github.com/RinktLtd/runner) to see what Rinkt can automate.
